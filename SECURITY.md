## Scope

This policy applies to:
- Server-side code execution
- Sandbox / tool execution logic
- API key handling and configuration
- Dependency-related security issues

It does **not** cover:
- Third-party service outages
- Misuse of the software outside documented use cases
- Issues caused by modified or forked versions
